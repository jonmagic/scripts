# Jonmagic Scripts Raycast Extension

Raycast commands for fast Brain actions.

## Commands

| Command | Description |
| --- | --- |
| `Create Daily Project Note` | Create a numbered note in `Daily Projects/YYYY-MM-DD/`. |
| `Capture Weekly Note` | Append a timestamped rough commitment with optional source under `## Captured` in the current weekly note. |
| `Weekly Focus` | Show up to five unchecked weekly-note TODOs, waiting items, and captured count. The primary action opens a new cmux workspace in `~/Brain` and starts Copilot with the selected TODO. |

`Weekly Focus` also has an action to open `weekly-focus-card` in a cmux workspace for a sparse full-screen-friendly view.
