# Jonmagic Scripts Raycast Extension

Raycast commands for fast Brain actions.

## Commands

| Command | Description |
| --- | --- |
| `Create Daily Project Note` | Create a numbered note in `Daily Projects/YYYY-MM-DD/`. |
| `Capture Weekly Note` | Append a timestamped rough commitment with optional source under `## Captured` in the current weekly note. |
| `Weekly Focus` | Open the native full-screen Weekly Focus app. |

The native app shows at most five TODOs, resizes to the current monitor, and can open a cmux Copilot workspace for the selected TODO.
